# Self-Registration (Auto-Inscription) & Activation PIN Codes

- **Registration Fields**:
  - `Code Établissement` (School Code): Represents the school ID or slug (e.g., `1` or `excellence`).
  - `Matricule` (Registration Number): For students, this is `num_admission` (e.g., `EDUT-2024-000003`). For teachers, this is `emp_id` or `email`.
  - `Code d'activation (PIN)`: Required to match the `activation_pin` field in database tables (`students` or `employees`).
- **Default / Initial PINs**:
  - For existing users, the default pin is pre-set to `123456` during database migration.
  - New students/teachers require their PIN code to be generated by the school admin and provided to them privately.
